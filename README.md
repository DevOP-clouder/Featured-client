# Featured Client

Featured Client is a non-cheat external launcher experience for Minecraft Pocket Edition, built from lessons inspired by open-source launchers.

## Vision
- Clean launcher UX
- Profile and pack management
- Compatibility checks for target game versions
- Diagnostics and safe launch flows
- Cosmetic framework using only original assets

## Current Status
Initial scaffold and planning documents are in progress.
